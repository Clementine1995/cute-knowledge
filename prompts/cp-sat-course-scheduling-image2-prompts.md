# CP-SAT 排课小助手 Image2 Prompt 包

## 使用说明

本文件对应 `drafts/cp-sat-course-scheduling-draft.md`。重点是生成能讲清 CP-SAT 机制的竖版信息图，不只是画可爱排课场景。

建议生成 5 页步骤版。重点不是把排课场景画可爱，而是让读者按页理解 CP-SAT 的工作顺序。

## 全局视觉设定

- 主角：可爱的排课小助手，圆润手绘角色，拿铅笔和夹板。
- 场景：大学课表、课程方块、老师图标、教室图标、时间槽。
- 固定风格：小红书 3:4 竖版信息图，软萌治愈 + 贴纸手账。奶油白纸张背景，低饱和浅粉、奶黄、薄荷绿，浅色描边，圆角贴纸，干净留白。
- 文字要求：每页只生成主标题、短副标题和 2-4 个短知识点。不要长段文字。

## 5 页步骤版 Prompt

### 第 1 页：总览封面

```text
生成一张小红书 3:4 竖版信息图。

主标题：CP-SAT 是什么？
副标题：先用排课看懂它

图中文字：
- 用排课理解
- 处理很多规则的安排算法
- 边试
- 边排除
- 记住死路
- 找方案

页面结构：
- 顶部：主标题和副标题，标题里的 CP-SAT 必须清楚可读。
- 中间：排课小助手站在大学课表前，手里拿着铅笔和错题本。
- 底部：先放一句短定义“处理很多规则的安排算法”，再放一条迷你流程带“边试 -> 边排除 -> 记住死路 -> 找方案”。

视觉内容：
课表旁边散落课程、老师、教室、时间的小贴纸。画面要让读者先看到“CP-SAT 是什么？”，再看到“用排课理解”和“处理很多规则的安排算法”。排课只是这次用来解释 CP-SAT 的例子，不要把 CP-SAT 表现成专门排课的软件。

固定风格：
软萌治愈 + 贴纸手账，奶油白纸张背景，低饱和浅粉、奶黄、薄荷绿，圆润手绘角色，浅色描边，圆角贴纸，干净留白。

禁止：
不要长段文字，不要乱码，不要水印，不要品牌 logo，不要复杂背景。
```
