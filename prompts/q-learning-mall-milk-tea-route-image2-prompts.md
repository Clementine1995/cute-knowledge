# Q-learning 商场找奶茶路线 Image2 Prompt 包

## 使用说明

本文件对应 `drafts/q-learning-mall-milk-tea-route-draft.md`。重点是生成能讲清 Q-learning 机制的竖版信息图，不只是画可爱商场地图。

建议生成 7 页步骤版。重点是让读者按页理解：看当前位置和可选动作，走一步拿反馈，把经验更新进 Q 值表，把后面的好处往前传一点，在熟路和新路之间平衡，最后形成“边试边记分”的循环。

## 全局视觉设定

- 主角：可爱的路线小助手，圆润手绘角色，拿小夹板和迷你地图。
- 场景：商场简化地图、路口格子、奶茶店目标图标、方向箭头、Q 值小表格。
- 固定风格：小红书 3:4 竖版信息图，软萌治愈 + 贴纸手账。奶油白纸张背景，低饱和浅粉、奶黄、薄荷绿，浅色描边，圆角贴纸，干净留白。
- 文字要求：每页只生成主标题、短副标题和 2-4 个短知识点。不要长段文字。
- 准确性边界：商场找奶茶路线只是本篇生活例子；Q-learning 的真实实现有更新公式、学习率、折扣因子，本篇只做简化理解。

## 7 页步骤版 Prompt

### 第 1 页：总览封面

```text
生成一张小红书 3:4 竖版信息图。

主标题：Q-learning 是什么？
副标题：用找奶茶路线看懂它

图中文字：
- 边走边学
- 给选择记分
- 简化理解

页面结构：
- 顶部：主标题和副标题，标题里的“Q-learning”必须清楚可读。
- 中间：一张软萌商场小地图，主角站在路口，远处有奶茶店图标。
- 右侧：迷你流程贴纸“看位置 -> 选方向 -> 得反馈 -> 改分数”。
- 底部：一句短边界提醒“这是简化理解”。

视觉内容：
商场地图用圆角格子表示，不要复杂真实商场平面图。路线小助手拿着迷你地图站在路口，有一点迷茫但不慌。画面要让读者先看到“Q-learning 是什么？”，再看到“边走边学”。不要画成自动导航软件广告。

固定风格：
软萌治愈 + 贴纸手账，奶油白纸张背景，低饱和浅粉、奶黄、薄荷绿，圆润手绘角色，浅色描边，圆角贴纸，干净留白。

禁止：
不要长段文字，不要乱码，不要水印，不要品牌 logo，不要复杂背景，不要过多装饰，不要画成科幻机器人，不要出现公式。
```
