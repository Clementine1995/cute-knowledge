# CP-SAT Image2 Prompt 包

## 使用说明

本文件先不锁死“宴会大管家”这个比喻。DeepSeek 的大管家只是一个参考，真正要保留的是 CP-SAT 的机制讲解：变量、约束、目标、传播、冲突学习、估算剪枝、全局约束、并行搜索。

当前建议先做“宴会排座”版本，因为它和已有 CP-SAT 样稿最接近；如果后续觉得不够好，可以换成“医院排班”或“旅行路线规划”版本。

## Image2 Prompt 生成原则

- 每页必须有 2-4 个短知识点。
- 不能只画可爱场景。
- 如果使用“大管家”，它只是视觉角色，不是固定账号 IP。
- 如果换比喻，机制词必须保留：变量、约束、目标、传播、冲突学习、剪枝。

## 4 页版 Prompt：宴会排座版

### 第 1 页：变量约束目标

```text
生成一张小红书 3:4 竖版信息图。

主标题：CP-SAT 到底在忙什么？
副标题：先把麻烦事分清楚

图中文字：
- 变量：谁坐哪
- 约束：不能破
- 目标：尽量好

页面结构：
- 顶部：标题区。
- 中间：宴会座位图，很多宾客和圆桌。
- 底部：变量、约束、目标三张卡片。

视觉内容：
一个可爱的调度员角色站在座位图前，手里拿夹板。画面表现“安排很多人和很多规则”，但不要拥挤。

固定风格：
软萌治愈 + 贴纸手账，奶油白背景，低饱和浅粉、奶黄、薄荷绿，圆润手绘小人，圆角贴纸，干净留白。

禁止：
不要长段文字，不要乱码，不要水印，不要复杂背景。
```

### 第 2 页：传播和冲突学习

```text
生成一张小红书 3:4 竖版信息图，沿用上一页风格。

主标题：它不是瞎试
副标题：会推理，也会记错题

图中文字：
- 传播：连锁后果
- 冲突：查原因
- 学习：记死路

页面结构：
- 顶部：标题区。
- 中间：左边是多米诺卡片，右边是后悔笔记本。
- 底部：调度员拿铅笔做标记。

视觉内容：
多米诺卡片从一个决定推到几个后果；旁边的小本本写“此路不通”。画面左右对比清楚。

固定风格：
软萌治愈 + 贴纸手账，奶油白背景，低饱和配色，浅色描边，圆角贴纸。

禁止：
不要长段文字，不要乱码，不要复杂背景。
```

### 第 4 页：并行和边界

```text
生成一张小红书 3:4 竖版信息图，沿用前面所有角色和风格。

主标题：所以它不是蛮力穷举

- 找高质量方案

- 中间：几个小调度员同时看不同座位图。

禁止：
不要长段文字，不要乱码，不要水印，不要复杂背景，不要强行画成科幻机器人。
```
